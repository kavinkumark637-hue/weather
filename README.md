**Real-Time Weather Application**

This is a web-based Weather Application developed using HTML, CSS, and JavaScript. The application allows users to check real-time weather information of any city by entering the city name.

**Live Website:**

https://weather-kavin4.vercel.app/

**Project Overview**

The main objective of this project is to fetch and display live weather data using an external API. The application takes user input, sends a request to the OpenWeatherMap API, and displays the weather details dynamically on the webpage

**Features:**

Search weather by city name

Displays temperature, humidity, and wind speed

Dynamic weather icons

Error handling for invalid input

Real-time date and time display

Responsive UI design

**Technologies Used:**

**HTML**

Used to create the structure of the webpage

**CSS**

Used for styling, layout, and responsive design

**JavaScript**

Used for logic, API calls, DOM manipulation, and event handling

**OpenWeatherMap API**

Used to fetch real-time weather data

**Vercel**

Used to deploy and host the website online

**How the Project Works**

User enters a city name in the input box

When the user clicks the search button or presses Enter:

JavaScript sends a request to the weather API using fetch()

The API returns data in JSON format

The application extracts required details such as:

Temperature

Humidity

Wind speed

Weather condition
The UI is updated dynamically using DOM manipulation
Weather icons are updated based on the weather condition
