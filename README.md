**Real-Time Weather App**

This project is a simple and responsive Weather Application developed using HTML, CSS, and JavaScript. It allows users to check real-time weather information by entering a city name.

The application uses the OpenWeatherMap API to fetch live weather data. When a user enters a city and clicks the search button or presses Enter, the app sends a request to the API and retrieves data in JSON format. The required information such as temperature, humidity, wind speed, and weather condition is extracted and displayed on the screen.

The user interface updates dynamically using JavaScript DOM manipulation. Weather icons change automatically based on the current weather condition. The application also includes error handling for invalid city names, API errors, and network issues.

Additionally, a real-time date and time feature is implemented using JavaScript, which updates every second to improve user experience.

**Features:**

Search weather by city name

Displays temperature, humidity, and wind speed

Dynamic weather icons

Error handling for invalid input

Real-time date and time display

Responsive UI design

**Technologies Used**
HTML for structuring the webpage

CSS for styling and layout

JavaScript for functionality and logic

OpenWeatherMap API for fetching live weather data
